Open Graph Test Page

Эта страница создана для тестирования Open Graph метатегов.

Картинка для превью: /images/og-image-final.jpg

Инструкции для тестирования:

  1. Скопируйте URL этой страницы: https://lavillapine.onrender.com/test-og.html
  2. Вставьте ссылку в Telegram, WhatsApp, или Facebook
  3. Проверьте что появилось правильное превью с вашей картинкой

Если превью не обновилось:

Некоторые платформы кэшируют превью. Попробуйте:

  • Facebook Sharing Debugger
  • Twitter Card Validator
  • Добавьте к URL параметр: ?cache=new
